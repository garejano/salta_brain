Criação das classes das entidades, mapping, repository, fixture.

Sugestão usar o prompt padrão, uma entidade de cada vez:

`Seguindo os padrões do projeto, crie a entidade, mapping, repository e fixture {NomeDaEntidade} ({TipoDeEntidade}), com: {TipoDeVariavel} {NomeDaVariavel} (...) Também faça os ajustes na entidade e mapping das classes relacionadas. Para o repositório, não se preocupe em criar métodos, apenas a classe e interface. Para o fixture, não atribua valores para os Ids das outras entidades. O fixture deve receber e instanciar as entidades diretamente, chamando o respectivo fixture para gerá-las quando a variável de entrada for null.`
