# EFC-5855 — Front - EstruturaPedagogica - Loop se usuário não tem acesso

## 1. Contexto e Motivação

Bug na tela de `EstruturaPedagogica`: quando o usuário não possui acesso à funcionalidade, a aplicação entra em loop ao invés de exibir uma mensagem de acesso negado ou redirecionar corretamente.

> A descrição no Jira contém texto de outro card (parece erro de preenchimento). O título é a referência mais confiável para o escopo deste bug.

Não especificado no card.

## 3. Subtarefas

Nenhuma subtarefa.

---

## 4. O Que Precisa Ser Feito

Investigar o fluxo de carregamento da tela `EstruturaPedagogica` quando o usuário não possui permissão de acesso, identificar onde ocorre o loop e corrigir o comportamento para exibir um estado de erro/acesso negado adequado.
