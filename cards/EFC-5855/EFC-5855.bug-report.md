# Bug Report — EFC-5855

> Gerado em: 2026-04-28

## Contexto

A tela de Estrutura Pedagógica possui controle de acesso por perfil. Ao tentar abrir qualquer página protegida por esse controle, o sistema verifica se o usuário logado possui as permissões necessárias antes de exibir o conteúdo.

## Descrição do Problema

**Comportamento observado:** Quando um usuário sem permissão de acesso tenta navegar para a área de Estrutura Pedagógica, a aplicação entra em loop contínuo — redirecionando o usuário repetidamente sem nunca exibir uma mensagem de erro ou página de acesso negado.

**Comportamento esperado:** O usuário deveria ser redirecionado para uma tela informando que não possui acesso àquela funcionalidade, ou ser levado de forma estável para a página inicial com uma mensagem clara.

## Passos para Reprodução

1. Fazer login com um usuário que **não** possui permissão de acesso à Estrutura Pedagógica.
2. Tentar acessar diretamente qualquer página da Estrutura Pedagógica (pela URL ou por um link no menu).
3. Observar que a aplicação fica redirecionando em loop sem apresentar nenhuma tela de erro ou mensagem ao usuário.

## Impacto

Afeta qualquer usuário que tente acessar a Estrutura Pedagógica sem ter o perfil adequado configurado. O loop impede que o usuário realize qualquer outra ação na aplicação até recarregar a página manualmente.

## Testes E2E

```gherkin
Feature: Controle de acesso à Estrutura Pedagógica

  Background:
    Given que o usuário está autenticado no sistema

  Scenario: Loop ao acessar Estrutura Pedagógica sem permissão (regressão)
    Given que o usuário não possui permissão de acesso à Estrutura Pedagógica
    When o usuário tenta navegar para qualquer página da Estrutura Pedagógica
    Then a aplicação não deve redirecionar o usuário em loop
    And o usuário deve visualizar uma mensagem informando que não possui acesso
    Or o usuário deve ser redirecionado de forma estável para a página inicial

  Scenario: Acesso permitido para usuário com permissão
    Given que o usuário possui permissão de acesso à Estrutura Pedagógica
    When o usuário navega para a página da Estrutura Pedagógica
    Then a página deve ser exibida normalmente
    And o usuário não deve ser redirecionado

  Scenario: Acesso via URL direta sem permissão
    Given que o usuário não possui permissão de acesso à Estrutura Pedagógica
    When o usuário tenta acessar a Estrutura Pedagógica digitando o endereço diretamente no navegador
    Then a aplicação deve exibir uma tela de acesso negado ou redirecionar para a página inicial
    And a aplicação deve permanecer estável, sem redirecionar em loop

  Scenario: Usuário sem nenhuma permissão no sistema
    Given que o usuário não possui nenhuma permissão configurada
    When o usuário tenta acessar a Estrutura Pedagógica
    Then a aplicação deve exibir uma mensagem de acesso negado
    And o usuário deve conseguir voltar à página inicial sem que a aplicação entre em loop
```
