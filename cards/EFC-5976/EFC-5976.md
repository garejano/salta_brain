# EFC-5976 — Autonomia de Avaliação e Cálculo para Itinerários

## 1. Contexto e Motivação

Como gestor pedagógico, quero configurar se o itinerário formativo utiliza regras próprias de avaliação e **se aparece junto ou separado no boletim**, para que o sistema represente corretamente o modelo pedagógico da escola sem depender da Formação Geral Básica (FGB).

O Novo Ensino Médio introduz Itinerários Formativos (IF) com ciclos e avaliações distintos do currículo regular. Atualmente o Configurador de Avaliações mistura ambos no mesmo grid, tornando o configurador difícil de usar e o boletim de leitura ruim.

> Plano de desenvolvimento detalhado, scripts SQL e análise do código estão em `../_EFC-5976/EFC-5976.md`.

Link Figma: https://www.figma.com/design/CYfFa216YlY5W0y7rIUgSO/Itinerar%C3%A1rios-Formativos?node-id=47-4680&t=pIBB6gvq1AOBSo29-1

---

## 2. Critérios de Aceitação

1. O sistema deve permitir configurar, para cada agrupamento, se o IF utiliza **estrutura de avaliação própria** ou **estrutura herdada/replicada da FGB**
2. Quando configurado com estrutura própria, o itinerário deve ser exibido como um "escopo" separado no configurador de avaliações com configuração própria
3. A configuração deve refletir de forma consistente em: boletim, ficha individual do aluno, relatórios acadêmicos e histórico escolar
4. Alterações na configuração **não devem ter efeito retroativo** sobre resultados já consolidados

## 4. O Que Precisa Ser Feito

- Separar itinerários/componentes formativos em escopos próprios no configurador de avaliações, com colunas independentes e filtragem correta de disciplinas
- Ampliar `EscopoEnum` com valores CF Anual / CF Semestral / CF Trimestral / CF Bimestral
- Adicionar FK `ItinerarioFormativoCicloId` em `EstruturaAvaliacao`
- Atualizar filtros de backend e front-end para suportar os novos escopos
- Alterar label "Itinerários são semestrais:" → "Exibir Componentes Formativos separadamente no boletim:"
